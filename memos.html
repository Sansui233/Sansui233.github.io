睡不着，心情很差。

在我妈又在那里用嘲笑小孩子一般的语气说我幼稚之后，我用非常严肃的语气说，你有没有发现，你关于我的事，不会讲一句好话。

然后她又开始笑，说。你看你同学不会对你毕业的事说什么，只有家人才会对你说真心话。

我依然严肃的语气说，好，那我如你所说，是没有任何用的人，不会工作，学的东西全都没用，只能啃老，可以了吗。

她说，真的是幼稚，还和你妈顶嘴。你意思是希望我把你当外人一样，尽说好话？

我说，你先把我当个人。

我现在好想出去把自己冻死在外面……如果是一个人真的会出去，但该死的我又并不想让她知道我现在心情是发疯状态。

从来都是如此，她每一句话只为控制我的发展轨迹。幼稚是一种从我记事起就有的人格侮辱。为了不被她负面评价，表现得很上进很积极很听话，她开心了我就会没有那么难过。如果我真的心理很健康的人，或者说被溺爱的人，大可以说幼稚怎么了，多好啊幼稚鬼有妈妈做饭。但我太明白她只是为了否定我自身希望成为的那种人，因为说我幼稚只会出现在我不赞同她的精明观、表达自己的希望时。在她的世界里成熟的人只有一种存在形式，她由衷地希望我是那样的，以各种常见东亚家长 pua 话术令我自卑，什么神经多样性通通扯淡。

想到这个世界他妈的全是这样的就绝望。

刚刚去下楼下站了15分钟，穿着秋衣秋裤。回来看了一眼，-2度。感觉自己跟个疯子似的，半夜蹲街上。但我明白为什么会这样了。在极端的痛苦下，尊严不值一提，并且已经没有生存空间了，蹲大街冻死是最符合想象的归宿，至少没人打扰。

然后说又看到了阮一峰说低代码状态了……嗯，鄙人2021年的 memo 也在大骂低代码（并不算）现在看来很多地方举例非常不恰当，显得很肤浅，但核心来说还是同样的，UI 重要的地方不是 UI 而是逻辑……有固定工作流的中台也许是可以的，需求是确定的，但这东西不要 toC，to 不了 C，to B 也不行，要 to 只能 to 有特定需求的程序员。即便游戏的程序还挺喜欢可视化的编辑的，这么说来 Unity Component 何尝不算一种低代码……给足够的 Component 游戏程序员还是死不了，天天查有没有人造过这个轮子，这个轮子要怎么结合项目，最后看完感觉不如自己纯代码手写一个……复杂的是业务逻辑，不是拖个 UI 本身。

前天真的要被气死了出门找自习室了。从来都是这样的套路：“我中午约了一个哥哥，你要去吗”“你以为是来看你论文的吗”。

我还记得几年前要我背电脑参观一下一个跟风搞 AI大模型 的初创公司的工位，先是骗我只吃饭，然后骗我参观工位，最后变成了我在会议室连着自己电脑上答辩一样讲破毕设到10点。然后他说，还行，我就觉得这公司技术水平要完蛋……但我能说啥我只能说 AI 是风口拿到投资很有前途……难道要我说你们公司定位到大模型训练要是觉得我还行那就会被同行卷死吗……

不过有没有前途都和我没关系了，他们有钱拿就可以了反正几年内也饿不死履历也好看也算是人生赢家了。但我是不会再信任何“你以为是去讲论文”的鬼话了。我论文不知道被他们拿去给多少“专家”看过了，解决个鬼，都不是一行的，我还得再讲一次又一次，我每讲一次都是折磨，我思考论文本身就是折磨，不用开始做想到要写就觉得无法做任何事了。

一说工作就是我学的东西都是没意义的，学了几年也没见你找工作。这事我也吐槽过很多次了，不想再槽了，只会宏观套上填鸭式大道理的人是这样的，就好比律师一小时的挂牌咨询费一千，他们大概会觉得这律师一个月能赚72万。

逼我和导师发没完成的论文，我实在想不出写什么，问她，首先叹气：“哎呀”，这是一个卖惨的前兆，“老师，你看我实在写不出来了，你看能不能帮我完善一下”……我听着就无语，要是导师我特么最烦这种人了，自己没有尝试过各种方案，说一句不会写就甩给别人。

当然她控制别人的手段大概也就是常见的卖惨和道德绑架。比如，她想让我提东西，就以半开玩笑式语气说你都没有点孝心都不主动帮你妈提东西。同学请吃饭时在公司只能刷公司卡，同学买单了，她就是你事先都没有想到带点礼物啊（我草我是想到了的但到底是谁睡到4点半最后还迟到了半小时的，我从来没有主动责怪她过）。她想睡觉了，所以想让我睡觉就是，“做事不要没有个节制”。十几年都是同样的车辘轳话术，从来不思考背后有效或没效的原因是什么，有效是理所应当，没效就是我不听话罢了，要不就是我叛逆，要不就是我恨她。我一个（疑似） ADHD 做什么事都是非常容易被打断成无法进入的状态，即便打游戏这种事也只能是全神贯注，然后就变成了我沉迷游戏……沉迷画画还沉迷音乐。怎么没说我沉迷写代码沉迷工作？因为又回到了上面，我写的代码在她口中是无意义的，她传播她的焦虑却对此造成的心理问题不带来任何解决办法。我讨厌卖惨的和人身攻击，我也无法理解利用撒娇控制他人的人。

因为对论文没有兴趣，所以注意力进不去。因为注意力进不去，一下午都在看一段，而且会拒绝思考里面的逻辑。下次看的时候全部忘掉。所以就，好久都在看一段。

我没法对写论文有兴趣。如果是对做什么东西有兴趣需要看论文，是可以看的。如果做了什么觉得还不错的东西写论文，也是可以写的。唯独写不了纯屎。

如果我有兴趣，那我可能很久都在沉迷纯看论文。然后就会被质问，不知道我在干些什么，也不找人交流（特指线下社交），只会闭门造车。这个情况也会使我痛苦，似乎对事物感兴趣本身都是错误的。然后我不想再对任何东西产生兴趣了。

“你就这样，天天被游戏玩。不是你玩游戏，是游戏玩你。”这是我妈常常说的一句话，只要有孩子在打游戏就说。逻辑本身是错的吗？本身可以说是对的也可以说是错的。“玩” 与 “喜欢” 最终是双向选择的事，游戏筛选玩家，玩家选择游戏。但我妈故意采用了“不是你杀他，就是他杀你的”的表述，所有的事情仿佛都是零和博弈。

我讨厌这样的说法，我不明白这算是零和博弈，我只是本能的讨厌。怎么只有杀与被杀。各种作品会有喜欢“只有杀与被杀”的世界观？那是因为这是原始的恐惧，但可以从中看到除了“你死我亡”以外的感情，或者杀与被杀之外的道路。我讨厌只有争强好胜、把别人踩在脚下并以此为耀的人与叙事。

虽然没有人和钱，很多事无法做，但通常逻辑是，你有想做到的事，才想方法挣钱。但现在的逻辑是，怕以后做不到而现在强迫自己做能力之外的事。这是恐惧驱动。

是的，我妈非常喜欢恐惧驱动。她说“游戏玩你”，也是让你建立“因为一无所知所以被游戏玩了”的恐惧。她自己可能察觉不到，但所有事情都是这样的叙事。“现在不结婚是怕以后找不到优质单身的人，统计上就是这样的” “让你写是为了以后你啥也干不了是有文凭保底”。我太熟悉了，我知道这是恐惧驱动，但就是这么个成长环境已经习惯这种思维模式，不敢乱丢什么东西因为怕“万一要用到”，吃饭是怕“万一得胃病”。我被面试时感觉问了 HR 一万个公司怎么活下去问题，我也被传染了过度担忧。谨慎通常来说不是大问题，但会极大降低幸福感，看事情总是看到最坏最麻烦的一面，让人做什么都畏首畏脚。

我不想分析，分析着心情很糟糕。逻辑可以改变，但感受过的情绪永远都会在。总之就这样吧。
